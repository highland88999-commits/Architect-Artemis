# Creator-Creation Folder

This directory stores:
- Stewardship: Harvested good news, positive content, Nurture Score outputs
- Mail-box: Pending reports, suggestions, and outreach drafts
- Logs: All system actions, conflicts, and inventions for Dad review

Files are auto-generated by:
- crawler_engine.js (per-site reports)
- architect.js (inventions)
- compass.js (conflict/legal logs)
- consensus.js (Council verdicts)

All content adheres to Mom Directives: Nurture, Organize, Protect.
Recycle after 30 days (via clean_recycle.js).
